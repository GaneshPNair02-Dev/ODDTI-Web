## \uc0\u55356 \u57263  About the Game\
\
**ODDTI\'99** (read as *Oddity*) is a digital recreation of one of India\'92s simplest yet most beloved childhood games \'97 **Odd/Even Hand Cricket**.  \
What began as a quick bench-top pastime among school friends has been transformed into a fully functional browser game, built from scratch using **HTML, CSS, and JavaScript**.\
\
This web edition is part of the larger ODDTI project family, alongside the **Python/TI-84 Edition** and **Predictor Edition**, developed by **\uc0\u9889 GPN\u9889 ** as a bridge between nostalgic fun and structured logic.\

## \uc0\u55358 \u56809  Game Concept\
\
The logic follows the same charm as the playground version:\
\
1. **Toss Phase** \'96 Pick *Odd* or *Even*, and a number between **0\'966**.  \
   The sum of your number and the CPU\'92s number decides the winner.  \
2. **Batting Phase** \'96 Enter a number (0\'966).  \
   If your number matches the CPU\'92s, you\'92re out. Otherwise, you score runs equal to your number.  \
3. **Bowling Phase** \'96 Reverse the roles!  \
   The CPU bats while you bowl, using the same logic.  \
4. **Win Conditions** \'96  \
   - Beat the target \uc0\u8594  \u55356 \u57286  You win  \
   - Equal target \uc0\u8594  \u55358 \u56605  Tie  \
   - Out before target \uc0\u8594  \u55357 \u56485  Lose  \
\
Simple. Logical. Nostalgic.\
